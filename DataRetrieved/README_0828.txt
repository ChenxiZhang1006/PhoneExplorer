storeDataIntoFile.sh

Get data from the database, and store them into the .txt file

java -jar storeDataIntoFile.jar filePath1 fileName1 filePath2 fileName2

filePath1/fileName1 where the variable desctiption file for variable expand, and its name.

filePath2/fileName2 where the variable information file which is used for index, and its name.
